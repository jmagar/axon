  Assistant:
HOT applies only when the update does not modify any columns referenced by the table’s indexes, including expression and partial indexes, and when there is sufficient free space on the page containing the old row for the updated row. [S1]

When those conditions are met, HOT reduces two update overheads: it avoids needing new index entries for the updated row, and it lets old versions of updated rows be removed during normal operation instead of requiring periodic vacuum operations. [S1]
