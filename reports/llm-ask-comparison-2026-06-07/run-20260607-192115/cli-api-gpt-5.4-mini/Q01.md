  Assistant:
Bun does not run arbitrary lifecycle scripts for installed dependencies by default because executing those scripts can be a security risk and could allow malicious code to run on your machine. [S2][S3][S4]

To allow a specific dependency’s lifecycle scripts to run, the project must add that package to `trustedDependencies` in `package.json` and then reinstall the package so Bun reads that field and runs its scripts. [S2][S3]
