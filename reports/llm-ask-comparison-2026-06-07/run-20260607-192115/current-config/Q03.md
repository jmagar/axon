Subnet routers are for reaching specific private subnets, such as office LANs, cloud VPCs, printers, cameras, legacy devices, or managed services that do not run the Tailscale client.[S1] They act as gateways between the tailnet and conventional subnet-based networks, advertise routes to those private IP ranges, and let tailnet devices access non-Tailscale devices behind the router.[S1]

Exit nodes are for routing outbound internet or non-Tailscale traffic from tailnet devices through a chosen tailnet device, effectively functioning like a VPN server.[S1][S8] When a device uses an exit node, its internet traffic appears to come from the exit node’s location, which is useful for privacy on untrusted networks or accessing location-dependent resources.[S1][S6] By default, exit nodes capture network traffic that is not already directed to a subnet router or app connector.[S8]

Practically, use a subnet router when the problem is “I need my tailnet users to reach this private network or device,” and use an exit node when the problem is “I want this tailnet device’s outbound internet traffic to leave through another device.”[S1][S8][S9] The docs also frame this as subnet routers routing inbound access to non-Tailscale devices, while exit nodes route outbound traffic.[S9]

SNAT changes what the destination sees when traffic comes from devices behind a subnet router.[S1] By default, Tailscale subnet routers use SNAT, so traffic from a device behind the subnet router appears to come from the subnet router itself rather than from the original device.[S1] This reduces routing complexity and enables more transparent communication, but it hides the original source IP and can limit scenarios that require traffic to traverse multiple networks.[S5][S10] If SNAT is disabled with `--snat-subnet-routes=false` on Linux, the original source IP is preserved, but return routing must be configured so devices know to send Tailscale traffic back through the subnet router, using `100.64.0.0/10` with the subnet router’s LAN IP as the next hop.[S1][S5]
