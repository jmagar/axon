  Assistant:
Special allocation classes are designated for storing metadata and, optionally, small file blocks [S12]. To add special-class devices, a pool must first have at least one normal (non-dedup/special) vdev present [S1]. Small-block placement is enabled by setting the `special_small_blocks` property on the dataset to a non-zero value, which indicates the size threshold for including small file blocks in the special class [S7][S8][S9][S14]. The redundancy of the special device should match that of the pool's other normal devices [S1].
