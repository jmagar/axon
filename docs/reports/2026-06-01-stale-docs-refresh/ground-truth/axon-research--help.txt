Web research via Tavily AI search with LLM synthesis

Usage
  axon research [OPTIONS] [TEXT]...

Arguments
  [TEXT]...          Text query or prompt

Options
  --limit <n>                      Maximum number of search results
  --research-depth <n>             Number of sources to synthesize over
  --search-time-range <range>      Restrict search to day, week, month, or year
  --json                           Output machine-readable JSON
  -h, --help                       Print help
