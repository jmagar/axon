# Android OAuth Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add Android-side OAuth sign-in for Axon using Axon's existing OAuth server while preserving static bearer-token support.

**Architecture:** Android will support `Bearer` and `OAuth` auth modes. OAuth uses AppAuth authorization-code + S256 PKCE, Axon's existing OAuth authorization-server metadata, dynamic registration, encrypted persisted AppAuth `AuthState`, single-flight token refresh, and mode-aware request decoration inside `AxonClient`. `/api/panel/*` remains explicit panel/server-config auth and must not be implicitly unlocked by OAuth access tokens in this plan.

**Tech Stack:** Kotlin 2.1.0, Android Gradle Plugin 8.13.2, minSdk 24, targetSdk 35, Jetpack Compose, OkHttp 4.12.0, kotlinx.serialization 1.7.3, AndroidX Security Crypto 1.1.0, AppAuth Android 0.11.1.

## Global Constraints

- Do not add or change Axon server OAuth implementation; Android consumes the existing `AXON_MCP_AUTH_MODE=oauth` flow.
- Preserve static bearer-token support for users who paste `AXON_MCP_HTTP_TOKEN`.
- Use internal redirect URI `com.axon.app://oauth2redirect` for this slice.
- Document that verified HTTPS App Links are preferred before broader production distribution because custom schemes can collide.
- Store OAuth refresh tokens only inside encrypted app storage; never DataStore plaintext.
- Exclude OAuth state storage from cloud backup and device transfer.
- Keep `android:allowBackup="false"` unchanged.
- OAuth tokens must be refreshed before REST and SSE calls; refresh once before opening an SSE stream, never per SSE event.
- OAuth refresh and `AuthState` mutation must be single-flight and protected by a `Mutex`.
- Persist `AuthState` only when AppAuth state changes; do not write encrypted prefs on every API call.
- Treat OAuth state persistence failure as a sign-in or refresh failure.
- Do not rely on a native-client secret. Android dynamic registration requests `token_endpoint_auth_method=none`, and token/refresh requests use no client authentication.
- The existing Android package/application id is `com.axon.app`.
- Existing Android build command remains `./gradlew -PaxonAuroraAndroidPath=/home/jmagar/workspace/aurora-design-system/android :app:assembleDebug`.
- Existing Android unit test command remains `./gradlew -PaxonAuroraAndroidPath=/home/jmagar/workspace/aurora-design-system/android :app:testDebugUnitTest`.
- Server-side operator config must allow `com.axon.app://oauth2redirect` in `AXON_MCP_AUTH_ALLOWED_REDIRECT_URIS`.

Create `apps/android/app/src/main/java/com/axon/app/data/auth/OAuthStateStore.kt`:

```kotlin
package com.axon.app.data.auth

import android.content.Context
import android.util.Log
import androidx.security.crypto.EncryptedSharedPreferences
import androidx.security.crypto.MasterKey

private const val TAG = "OAuthStateStore"
private const val PREFS_NAME = "axon_oauth_state"
private const val KEY_AUTH_STATE = "auth_state_json"

class OAuthStateStore(context: Context) {
    private val appContext = context.applicationContext

    private fun createPrefs() = EncryptedSharedPreferences.create(
        appContext,
        PREFS_NAME,
        MasterKey.Builder(appContext)
            .setKeyScheme(MasterKey.KeyScheme.AES256_GCM)
            .build(),
        EncryptedSharedPreferences.PrefKeyEncryptionScheme.AES256_SIV,
        EncryptedSharedPreferences.PrefValueEncryptionScheme.AES256_GCM,
    )

    fun read(): String? = runCatching {
        createPrefs().getString(KEY_AUTH_STATE, null)?.takeIf { it.isNotBlank() }
    }.recoverCatching { cause ->
        Log.w(TAG, "OAuth state read failed; clearing corrupted encrypted store", cause)
        clear()
        null
    }.getOrNull()

    fun write(rawJson: String): Boolean = runCatching {
        createPrefs().edit().putString(KEY_AUTH_STATE, rawJson).commit()
    }.onFailure {
        Log.w(TAG, "OAuth state write failed", it)
    }.getOrDefault(false)

    fun clear(): Boolean = runCatching {
        createPrefs().edit().remove(KEY_AUTH_STATE).commit()
    }.onFailure {
        Log.w(TAG, "OAuth state clear failed", it)
    }.getOrDefault(false)
}
```

Create `apps/android/app/src/main/java/com/axon/app/data/auth/OAuthRepository.kt`:

```kotlin
package com.axon.app.data.auth

import android.content.Context
import android.content.Intent
import android.net.Uri
import kotlinx.coroutines.Dispatchers
import kotlinx.coroutines.sync.Mutex
import kotlinx.coroutines.sync.withLock
import kotlinx.coroutines.suspendCancellableCoroutine
import kotlinx.coroutines.withContext
import net.openid.appauth.AuthState
import net.openid.appauth.AuthorizationException
import net.openid.appauth.AuthorizationRequest
import net.openid.appauth.AuthorizationResponse
import net.openid.appauth.AuthorizationService
import net.openid.appauth.AuthorizationServiceConfiguration
import net.openid.appauth.RegistrationRequest
import net.openid.appauth.RegistrationResponse
import net.openid.appauth.ResponseTypeValues
import kotlin.coroutines.resume

private val AXON_OAUTH_REDIRECT_URI: Uri = Uri.parse("com.axon.app://oauth2redirect")
private const val AXON_OAUTH_SCOPE = "axon:read axon:write offline_access"

class OAuthRepository(
    context: Context,
    private val stateStore: OAuthStateStore,
) : OAuthTokenSource {
    private val service = AuthorizationService(context.applicationContext)
    private val mutex = Mutex()
    private var authState: AuthState? = null
    private var signInInFlight = false

    override fun isSignedIn(): Boolean = runCatching { loadedState().isAuthorized }.getOrDefault(false)

    suspend fun createAuthorizationRequest(baseUrl: String): Intent = withContext(Dispatchers.IO) {
        mutex.withLock {
            if (signInInFlight) error("OAuth sign-in already in progress")
            signInInFlight = true
        }
        runCatching {
            val serviceConfig = discover(baseUrl)
            val registration = register(serviceConfig)
            val request = AuthorizationRequest.Builder(
                serviceConfig,
                registration.clientId,
                ResponseTypeValues.CODE,
                AXON_OAUTH_REDIRECT_URI,
            )
                .setScope(AXON_OAUTH_SCOPE)
                .build()
            mutex.withLock {
                val state = AuthState(serviceConfig).also { it.update(registration) }
                persistOrThrow(state)
                authState = state
            }
            service.getAuthorizationRequestIntent(request)
        }.onFailure {
            mutex.withLock { signInInFlight = false }
        }.getOrThrow()
    }

    suspend fun handleAuthorizationResponse(intent: Intent): Result<Unit> = withContext(Dispatchers.IO) {
        val response = AuthorizationResponse.fromIntent(intent)
        val exception = AuthorizationException.fromIntent(intent)
        if (response == null) {
            mutex.withLock { signInInFlight = false }
            return@withContext Result.failure(exception ?: IllegalStateException("OAuth response missing"))
        }
        suspendCancellableCoroutine { continuation ->
            service.performTokenRequest(response.createTokenExchangeRequest()) { tokenResponse, tokenException ->
                val result = runCatching {
                    val state = loadedState()
                    state.update(tokenResponse, tokenException)
                    if (tokenResponse == null) throw tokenException ?: IllegalStateException("OAuth token exchange failed")
                    persistOrThrow(state)
                    authState = state
                }
                signInInFlight = false
                continuation.resume(result)
            }
        }
    }

    override suspend fun freshAccessToken(): Result<String> = withContext(Dispatchers.IO) {
        mutex.withLock {
            val state = loadedState()
            val before = state.jsonSerializeString()
            suspendCancellableCoroutine { continuation ->
                state.performActionWithFreshTokens(service) { accessToken, _, exception ->
                    val result = runCatching {
                        if (accessToken.isNullOrBlank()) throw exception ?: MissingAuthException("OAuth access token unavailable")
                        val after = state.jsonSerializeString()
                        if (after != before) persistOrThrow(state)
                        authState = state
                        accessToken
                    }
                    continuation.resume(result)
                }
            }
        }
    }

    fun signOut(): Boolean {
        authState = AuthState()
        signInInFlight = false
        return stateStore.clear()
    }

    fun dispose() {
        service.dispose()
    }

    private fun loadedState(): AuthState {
        authState?.let { return it }
        val loaded = stateStore.read()?.let { AuthState.jsonDeserialize(it) } ?: AuthState()
        authState = loaded
        return loaded
    }

    private fun persistOrThrow(state: AuthState) {
        if (!stateStore.write(state.jsonSerializeString())) {
            throw IllegalStateException("Could not securely store OAuth credentials")
        }
    }

    private suspend fun discover(baseUrl: String): AuthorizationServiceConfiguration =
        suspendCancellableCoroutine { continuation ->
            val metadataUrl = Uri.parse("${baseUrl.trimEnd('/')}/.well-known/oauth-authorization-server")
            AuthorizationServiceConfiguration.fetchFromUrl(metadataUrl) { config, exception ->
                continuation.resumeWith(
                    if (config != null) Result.success(config)
                    else Result.failure(exception ?: IllegalStateException("OAuth discovery failed"))
                )
            }
        }

    private suspend fun register(config: AuthorizationServiceConfiguration): RegistrationResponse =
        suspendCancellableCoroutine { continuation ->
            val request = RegistrationRequest.Builder(config, listOf(AXON_OAUTH_REDIRECT_URI))
                .setApplicationType("native")
                .setTokenEndpointAuthenticationMethod("none")
                .build()
            service.performRegistrationRequest(request) { response, exception ->
                continuation.resumeWith(
                    if (response != null) Result.success(response)
                    else Result.failure(exception ?: IllegalStateException("OAuth registration failed"))
                )
            }
        }
}

internal fun redactOAuthError(raw: String): String =
    raw.replace(Regex("(?i)(access_token|refresh_token|code|client_secret)=([^&\\s]+)"), "$1=<redacted>")
```

package com.axon.app.data.auth

import androidx.test.core.app.ApplicationProvider
import kotlinx.coroutines.async
import kotlinx.coroutines.awaitAll
import kotlinx.coroutines.runBlocking
import org.junit.Assert.assertFalse
import org.junit.Assert.assertTrue
import org.junit.Test
import org.junit.runner.RunWith
import org.robolectric.RobolectricTestRunner

@RunWith(RobolectricTestRunner::class)
class OAuthRepositoryTest {
    @Test
    fun `redactOAuthError removes oauth secrets`() {
        val raw = "access_token=a refresh_token=b code=c client_secret=d"
        val redacted = redactOAuthError(raw)
        assertFalse(redacted.contains("=a"))
        assertFalse(redacted.contains("=b"))
        assertFalse(redacted.contains("=c"))
        assertFalse(redacted.contains("=d"))
        assertTrue(redacted.contains("access_token=<redacted>"))
    }

        assertTrue(store.write("""{"config":{}}"""))
        val repo = OAuthRepository(context, store)

        assertTrue(repo.signOut())
        assertFalse(repo.isSignedIn())
    }

    @Test
    fun `ten concurrent freshAccessToken calls are single flight`() = runBlocking {
        val context = ApplicationProvider.getApplicationContext<android.content.Context>()
        val store = OAuthStateStore(context)
        val repo = OAuthRepository(context, store)

        val results = (1..10).map {

private suspend fun authRequest(builder: Request.Builder, panelRoute: Boolean = false): Request.Builder {
    val (_, auth) = config.get()

}

suspend fun beginOAuthSignIn(serverUrl: String): Result<Intent> = runCatching {
    if (_oauthStatus.value == OAuthUiStatus.Starting) error("OAuth sign-in already in progress")
    _oauthStatus.value = OAuthUiStatus.Starting
    val trimmedServerUrl = serverUrl.trim()
    validateAxonServerUrl(trimmedServerUrl)
    container.oauthRepository.createAuthorizationRequest(trimmedServerUrl)
}.onFailure {
    _oauthStatus.value = OAuthUiStatus.Error
}

fun completeOAuthSignIn(intent: Intent?) {
    viewModelScope.launch {
        if (intent == null) {
            _oauthStatus.value = OAuthUiStatus.Error
            _saveState.value = SaveState.Failed("OAuth sign-in was cancelled")
            return@launch
        }
        container.oauthRepository.handleAuthorizationResponse(intent).fold(
            onSuccess = {
                val updated = _settings.value.copy(authMode = AuthMode.OAuth)
                container.settingsRepository.save(updated)
                container.applySettings(updated.serverUrl.value, updated.token.value, updated.authMode)
                _settings.value = updated
                _draftAuthMode.value = AuthMode.OAuth
                _oauthStatus.value = OAuthUiStatus.SignedIn
                _saveState.value = SaveState.Saved
            },
            onFailure = { cause ->
                _oauthStatus.value = OAuthUiStatus.Error
                _saveState.value = SaveState.Failed(cause.message ?: "OAuth sign-in failed")
            },
        )
    }
}

fun cancelOAuthSignIn() {
    _oauthStatus.value = OAuthUiStatus.Error
    _saveState.value = SaveState.Failed("OAuth sign-in was cancelled")
}

fun signOutOAuth() {
    viewModelScope.launch {
        val cleared = container.oauthRepository.signOut()
        if (!cleared) {
            _saveState.value = SaveState.Failed("Could not clear OAuth credentials")
            return@launch
        }
        val updated = _settings.value.copy(authMode = AuthMode.Bearer)
        container.settingsRepository.save(updated)
        container.applySettings(updated.serverUrl.value, updated.token.value, updated.authMode)
        _settings.value = updated
        _draftAuthMode.value = AuthMode.Bearer
        _oauthStatus.value = OAuthUiStatus.Idle
        _saveState.value = SaveState.Saved
    }
}

## Failure Modes Table

| Codepath | Failure Mode | Rescued? | Test? | User Sees? | Logged? |
|---|---|---:|---:|---|---:|
| Redirect receiver | Another app claims custom scheme or manifest route is wrong | Y | Y | OAuth sign-in error/no callback | Y |
| Metadata discovery | Server metadata URL differs from OIDC issuer discovery | Y | Y | OAuth discovery failed | Y |
| Dynamic registration | `/register` unavailable or rejects redirect URI | Y | Y | OAuth registration failed | Y |
| OAuth callback | Wrong state, missing code, replay, callback after sign-out | Y | Y | OAuth sign-in failed | Y |
| OAuth state storage | Encrypted prefs corrupted or write fails | Y | Y | Sign in again / storage failure | Y |
| Token refresh | Concurrent requests refresh at once | Y | Y | At most one visible refresh failure | Y |
| REST auth | OAuth mode has no static bearer token | Y | Y | Request succeeds or sign-in-again | Y |
| SSE auth | Stream uses stale static bearer headers | Y | Y | Stream succeeds or auth error | Y |
| Panel routes | OAuth JWT sent to `/api/panel/*` | Y | Y | Server config requires bearer/panel auth | Y |
| Settings UI | User cancels browser or taps Sign in repeatedly | Y | Y | Cancelled/in-progress feedback | Y |
| Sign-out | Encrypted clear fails | Y | Y | Could not clear OAuth credentials | Y |
| Backup/transfer | Refresh state restored to another context | Y | Y | State excluded from backup/transfer | N |

No row has `RESCUED=N` and `TEST=N` with silent user impact.
