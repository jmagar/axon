# Parsing
Last Modified: 2026-07-15

Parsing extracts structured facts from prepared source documents.

## Parser Outputs

Parsers may produce symbols, headings, links, dependencies, package metadata,
code facts, tables, or graph candidates. Parser output should include
provenance back to document and source item identity.

## Rule

Parsers enrich source records. They do not acquire content and do not publish
vectors directly.
