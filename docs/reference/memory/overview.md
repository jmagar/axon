# Memory Overview
Last Modified: 2026-07-15

Axon memory stores durable facts, decisions, and context for later retrieval.

## Model

Memories are source-backed records with identifiers, content, metadata,
provenance, graph edges, and optional supersession relationships.

## Operations

Memory operations include remembering, searching, showing, linking,
superseding, reviewing, and building context for an agent or user task.

## Retrieval

Memory retrieval should use the same safety, freshness, redaction, and source
identity rules as other indexed content.
