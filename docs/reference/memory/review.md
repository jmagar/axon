# Memory Review
Last Modified: 2026-07-15

Memory review is the process of validating, correcting, superseding, or
retiring durable memories.

## Review Outcomes

| Outcome | Meaning |
|---|---|
| keep | memory remains active |
| correct | memory is updated with better evidence |
| supersede | a newer memory replaces it |
| retire | memory should not be retrieved by default |

## Requirements

Review actions must preserve auditability. They should record who reviewed the
memory, when it was reviewed, and what evidence justified the result.
