# Presentation Surface
Last Modified: 2026-07-15

Presentation docs cover colors, symbols, tokens, and generated UI/CLI
presentation assets.

## Generated Assets

Presentation tokens and schemas are generated by the presentation xtask. Check
the generated files under `docs/reference/presentation/` for the current token
snapshot.

## Rule

Presentation changes must keep CLI, web, Palette, and other clients aligned
with the shared presentation contract.
