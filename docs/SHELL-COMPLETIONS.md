# Shell Completions
Last Modified: 2026-03-10

Axon can generate shell completion scripts directly from the CLI:

```bash
axon completions bash
axon completions zsh
axon completions fish
```

Installation targets:

- Bash: `~/.local/share/bash-completion/completions/axon`
- Zsh: `~/.zfunc/_axon`
- Fish: `~/.config/fish/completions/axon.fish`

See [`docs/commands/completions.md`](./commands/completions.md) for command syntax and per-shell install steps.
