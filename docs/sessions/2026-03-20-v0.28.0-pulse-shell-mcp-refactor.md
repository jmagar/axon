## Key Findings

- `crates/core/logging.rs:73–95` — TOCTOU races in `SizeRotatingFile` rotation eliminated: `exists()`+`rename/remove` patterns replaced with match-on-`io::ErrorKind::NotFound`.
- `crates/core/config/parse/build_config.rs` — Four identical `split(',').map(trim).filter(empty).collect()` patterns collapsed into generic `parse_csv_env<F, T>` helper.
- `crates/mcp/server/handlers_embed_ingest.rs` — `parse_ingest_source` was mutating `&mut IngestRequest` via `.take()`; refactored to accept decomposed positional params, eliminating struct-borrow coupling.
- `crates/mcp/server/common.rs` — `validate_mcp_urls` now includes array index in error messages (`urls[{i}]: {e}`); `parse_job_id` changed from `Option<&String>` to `Option<&str>`.
- Pre-commit `rustfmt` hook catches line-wrapping differences (single-line vs multi-line fn signatures) — these are not semantic changes, just formatting.
- 1439 tests passing post-commit (confirmed by `test` hook in lefthook output).
