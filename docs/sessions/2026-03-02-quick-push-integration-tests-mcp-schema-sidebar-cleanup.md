## Technical Decisions

- **`_` prefix for unused destructured vars** (`_placeholderVisible`, `_currentMode`) — standard Rust/TS convention for intentionally-unused variables; cleaner than removing from destructuring (keeps the hook API shape visible).
- **`#[cfg(test)] use constants::ALLOWED_FLAGS`** — separate cfg-gated import rather than adding to the non-test use statement, avoiding the "unused import in lib target" error.
- **`div_ceil()` over manual implementation** — Rust 1.73+ stabilized `usize::div_ceil()`; Clippy correctly flags the `(n + d - 1) / d` pattern.
- **`std::iter::repeat_n()` over `repeat().take()`** — Rust 1.82+ provided `repeat_n`; Clippy `iter_repeat_x` lint fires when `.take()` is used on `repeat()`.
- **SSH remote** — switched permanently in this session; future pushes will use SSH.
